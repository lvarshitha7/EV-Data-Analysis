Electric Vehicle Data Analysis Dashboard

📊 Project Overview

This project analyzes Electric Vehicle (EV) adoption trends using interactive data visualization. The dashboard provides insights into EV distribution by model year, manufacturer, state, CAFV eligibility, and vehicle type.

The goal of this project is to explore EV growth patterns and understand which manufacturers and regions are leading the adoption of electric vehicles.

🛠 Tools & Technologies Used

Power BI – Dashboard creation and visualization

Excel / CSV Dataset – Data source

Data Cleaning & Transformation – Power Query

Data Visualization – Interactive charts and maps
